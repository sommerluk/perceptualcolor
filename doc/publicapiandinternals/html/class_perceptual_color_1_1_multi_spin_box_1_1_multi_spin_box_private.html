◆ isCursorPositionAtCurrentSectionValue()

bool PerceptualColor::MultiSpinBox::MultiSpinBoxPrivate::isCursorPositionAtCurrentSectionValue ( const int  cursorPosition) const

Test if a cursor position is at the current value.

Everything from the cursor position exactly before the value itself up to the cursor position exactly after the value itself. Example: “ab12cd” (prefix “ab”, value 12, suffix “cd”). The cursor positions 2, 3 and 4 are at the current value.

Parameters
cursorPositionthe cursor position to test
Returns
true if the indicated cursor position is at the value text of the current section. false otherwise.

◆ m_validator

QPointer<ExtendedDoubleValidator> PerceptualColor::MultiSpinBox::MultiSpinBoxPrivate::m_validator

The validator for the QLineEdit.

This validator allows changes only to the current section.

If the current section changes, also this validator’s configuration will be adapted to cover the new current section.

Note
It is not possible to change various values at the same time, for example by marking all the current text and use Ctrl-V to past a complete new value from the clipbord. This would be impossible to parse reliably, because the prefixes and suffixes of each section might contain (localized) digits that would be difficult to differenciate from the actual value.
