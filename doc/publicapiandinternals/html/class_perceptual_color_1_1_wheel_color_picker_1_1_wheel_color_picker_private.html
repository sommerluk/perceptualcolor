slot

Repaint m_colorWheel when focus changes on m_chromaLightnessDiagram

m_chromaLightnessDiagram is the focus proxy of m_colorWheel. Both show a focus indicator when keyboard focus is active. But apparently m_colorWheel does not always repaint when focus changes. Therefore, this slot can be connected to the qApp’s focusChanged() signal to make sure that the repaint works.

Note
It might be an alternative to write an event filter for m_chromaLightnessDiagram to do the same work. The event filter could be either WheelColorPicker or WheelColorPickerPrivate (the last case means that WheelColorPickerPrivate would still have to inherit from QObject). QSizeF PerceptualColor::WheelColorPicker::WheelColorPickerPrivate::optimalChromaLightnessDiagramSize ( ) const

Calculate the optimal size for the inner diagram.

Returns
The maximum possible size of the diagram within the inner part of the color wheel. With floating point precision. Measured in device-independant pixels.

The outer dimensions of the widget are a rectangle within a circumscribed circled, which is the inner border of the color wheel.

The widget size is composed by the size of the diagram itself and the size of the borders. The border size is fixed; only the diagram size can vary.

Known variables:

variable comment value
r relation b ÷ a maximum lightness ÷ maximum chroma
h horizontial shift left + right diagram border
v vertial shift top + bottom diagram border
d diameter of circumscribed circle inner diameter of the color wheel
b diagram height a × r
widgetWidth widget width a + h
widgetHeight widget height b + v
a diagram width ?

◆ resizeChildWidgets()

void PerceptualColor::WheelColorPicker::WheelColorPickerPrivate::resizeChildWidgets ( )

Update the geometry of the child widgets.

This widget does not use layout management for its child widgets. Therefore, this function should be called on all resize events of this widget.

Postcondition
The geometry (size and the position) of the child widgets are adapted according to the current size of this widget itself.
