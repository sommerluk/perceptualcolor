◆ m_gradientImageCache

GradientImage PerceptualColor::GradientSlider::GradientSliderPrivate::m_gradientImageCache

Cache for the gradient image.

Holds the current gradient image (without the handle). Always at the left is the first color, always at the right is the second color. This is idependent from the actual orientation and the actual LTR or RTL layout. So when painting, it might be necessary to rotate or mirror the image.
